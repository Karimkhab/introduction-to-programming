# Assignment 4: Ecosystem Simulation

A circular list of animals (Lion, Zebra, Boar) live on a field with growing grass. Each day animals may eat (grass or prey), make sounds, and lose energy. The simulation runs for a fixed number of days with strict validation and ordered exception messages.

## Input / Output

- Reads from `input.txt` and prints to standard output.
- First three lines: number of days, grass amount, number of animals.
- Next N lines: `Type Weight Speed Energy`.
