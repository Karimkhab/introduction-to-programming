# Introduction to Programming (Innopolis University)

Course assignments from the "Introduction to Programming" class (C + Java). Each assignment is stored in its own folder with the original solution and supporting diagrams/screenshots.

## Assignments

| # | Title | Language | Solution | Folder |
| - | ----- | -------- | -------- | ------ |
| 1 | Based Sum | C | `assignment-1-based-sum/solution.c` | `assignment-1-based-sum/` |
| 2 | Moodle Replacement | C | `assignment-2-moodle-replacement/solution.c` | `assignment-2-moodle-replacement/` |
| 3 | Smart Home System | Java | `assignment-3-smart-home-system/SmartHomeManagementSystem.java` | `assignment-3-smart-home-system/` |
| 4 | Ecosystem Simulation | Java | `assignment-4-ecosystem-simulation/Main.java` | `assignment-4-ecosystem-simulation/` |

## Overview

- Assignment 1: Validate numbers across multiple bases, apply index-based arithmetic, then sum results.
- Assignment 2: Command-driven course management with structs, unions, enums, and strict validation rules.
- Assignment 3: Smart home device management with OOP design, device states, and command processing.
- Assignment 4: Multi-day ecosystem simulation with grass growth, hunting rules, and ordered exceptions.
